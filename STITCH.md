# STITCH DESIGN INTEGRATION

This project follows the design template defined on Stitch:
https://stitch.withgoogle.com/projects/11149123587358401563

## Key Styles Applied
- **Colors**: Hex values taken from the dark imperial design (`#131313` background, `#d4af37` gold accents).
- **Aesthetic**: Ancient scroll metadata styling.
- **Font**: Google Fonts' `MedievalSharp` used for all headers, titles, and stats readouts.
- **Visuals**: Gold-accented containers, borders (`rgba(212, 175, 55, 0.2)`), clean custom button styles, and interactive elements.
- **Dynamic Alignments**: Implemented dynamic text alignments for Arabic books and support for system Right-To-Left (RTL) mirroring toggles in settings.
